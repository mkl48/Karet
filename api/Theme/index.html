Theme

One semantic colorscheme, piped through every Karet surface (terminal, Kaveat editor, autocomplete, pickers, windows). A theme is not a set of raw frame colours, it's a set of semantic highlight groups (Group) the way a Neovim colorscheme is, so Theme.Set("gruvbox") recolours the whole UI at once.

Authors may define a partial palette: any group left out is derived from the ones present (Surface lightens Background, Prompt follows Accent, ...), so a minimal theme still produces a complete, coherent Palette. Reach it as Karet.Theme. 

Karet.Theme.Define("midnight", { Background = Color3.fromHex("0b0e14"), Accent = Color3.fromHex("7aa2f7") })
Karet.Theme.Set("midnight")

Types​

Group​

</>
type Group = string

One semantic highlight group. The full set covers surfaces (Background, Surface, Foreground, Muted, Border, Selection), accents (Accent, AccentAlt), syntax (Command, Keyword, String, Number, Comment, Operator, Variable, Type, ...), diagnostics (Error, Warn, Info, Hint, Success) and terminal roles (Prompt, Output, ...). See Theme.Groups for the canonical list.

Palette​

</>
type Palette = {[Group]: Color3}

A map from Group to colour. May be partial when defining a theme (missing groups are derived); Theme.Get always returns a complete one.

Functions​

Define​

</>
Theme.Define(
name: string,
palette: Palette
) → Palette

Define (or redefine) a named theme from a partial palette. Returns the resolved complete palette, so you can pass it straight to Theme.Set or inspect it. Redefining the active theme recolours live.

Set​

</>
Theme.Set(theme: string | Palette) → ()

Set the active theme by name or by an explicit (partial) palette. Switching fires Theme.OnChanged, so every subscribed surface recolours at once. Errors if a name isn't defined.

Get​

</>
Theme.Get(group: Group?) → Palette | Color3

Read the active palette, or a single group's colour. Theme.Get() returns the full (always complete) palette; Theme.Get("Error") returns that group's Color3.

Name​

</>
Theme.Name() → string?

The name of the active theme, or nil if a raw palette was Theme.Set directly.

List​

</>
Theme.List() → {string}

Names of all defined themes.

Groups​

</>
Theme.Groups() → {Group}

The canonical list of group names (a copy, so callers can't mutate the source).

OnChanged​

</>
Theme.OnChanged(fn: (
palette: Palette,
name: string?
) → ()) → () → ()

Subscribe to theme changes. The callback fires immediately with the current theme and again on every Theme.Set or redefine. Returns an unsubscribe function. This is how every Karet surface recolours live.
