Terminal

This item only works when running on the client. Client

The visible console surface: window chrome, the scrolling output log, the input row, plus the toast stack and the live HUD. Driven entirely from the active Theme (recolours live). You get the active one back from Karet.Start on the client, or as Karet.Active.

Two presentations: the full window (default) and a single-line Compact command bar.
